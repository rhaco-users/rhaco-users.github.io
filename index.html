「シンプル＆パワフルなPHPライブラリ rhacoを使ってみよう！」
これから rhaco を使ってみようという方は、参考にしてみてください。

rhaco は PHPで書かれたオープンソースのライブラリです。MVC や MTV などの見通しの良い設計でアプリケーションの開発をすることが出来ます。PEAR などの外部ライブラリに一切依存せず、rhaco を一度設置するだけで、 O/R マッピング、テンプレートエンジン、そして共通のセットアップインタフェイスなど Web アプリケーションの開発を強力にサポートします。rhaco は、PHPのバージョン4 と 5 の両方で動作します。*1

シンプルで多機能なO/Rマッピング

xml 形式でテーブルモデルの定義を行うと、自動で対応するモデルを生成します。DB 操作 API は非常に多機能で、もう通常のアプリケーションの開発で SQL を書くことはありません。さらに、Views と組み合わせ、数行のコードで CRUD を実現出来ます。もう使い物にならない scaffolding とはお別れしましょう。

標準のアプリケーションセットアップ

作成したアプリケーションには、アプリケーションの設定やデータベース操作などが行える管理インタフェースが付属します。アプリケーションの設置で設定ファイルを操作することなく、全てブラウザ上でセットアップが完了します。

URL 設計も思いのまま

クールな URL も自由自在です。あなたの思いのままに URL マッピングを記述できます。さらに、付属のセットアップにより htaccess ファイルも自動で生成できます。

テンプレートエンジン

テンプレートの継承を実装したテンプレートエンジンで、ページレイアウトも簡単です。include まみれのテンプレートに嫌気がさしている方にぴったりです。

テンプレートエンジンに使われている XML ライブラリは、既存の HTML をスクレイピングするのにも有効です。PHP4 でも動作する XML パーサーとして使用するのも良いでしょう。

国際化対応

アプリケーションの国際化に gettext などの拡張は必要ありません。セットアップでアプリケーション内から翻訳の必要な箇所を抜き出し、gettext のファイルを編集する要領で簡単に国際化が実現出来ます。

アプリケーション

BlogKit
jaja で公開されている、シンプルなブログツール。rhaco の作者 tokushima さんのブログにも使用されています。
Conveyor
jajaアプリ。Plagger と同様に様々なプラグインで RSS フィードを処理することができます。すべての設定をブラウザ上で行うことができるのも特徴です。(リポジトリ)
Sayist
blog.riaf.jp で使用されているブログツール。BlogKit に比べ、多少機能が多くなっています。
Phwittr
PHP Framework Fight! のイベントで作成された Twitter ライクなアプリケーション。
